## 🚀 What this app does

GVRT-Trading-Bot is a Windows app for GRVT perpetual trading. It helps you manage more than one account and run three separate trading methods from one place.

It is built for users who want a simple way to:

- connect multiple accounts
- set a proxy for each account
- run one, two, or three strategies at the same time
- keep trading work in one desktop app

## 🖥️ What you need

Use a Windows PC with:

- Windows 10 or Windows 11
- 4 GB RAM or more
- 200 MB of free disk space
- A stable internet connection
- One or more GRVT accounts

If you plan to use many accounts, a stronger PC helps keep the app smooth.

## ⚙️ First-time setup

When you open the app for the first time:

1. Add your GRVT account details.
2. Set a proxy for each account if you use one.
3. Pick the trading strategy you want for each account.
4. Save your settings.
5. Start the bot.

Keep each account in its own profile so the app can handle them separately.

## 🧭 Main parts of the app

### 👤 Account manager

Use this area to add and edit accounts. You can keep each account separate and attach a proxy if needed.

### 🧪 Strategy panel

Choose from three built-in trading methods. Each one can run on its own account or be grouped with others.

### 🌐 Proxy settings

Set one proxy per account to help keep traffic separate. This is useful when you run many accounts.

### ▶️ Start and stop controls

Use the main buttons to start or stop trading for one account or for all accounts.

### 📊 Activity view

Check the latest bot activity, including account status, strategy status, and connection state.

## 🧩 How to use it

1. Open GVRT-Trading-Bot.
2. Add your first account.
3. Add a proxy if you use one.

## 🔒 Proxy setup tips

If you use a proxy, keep these points in mind:

- Use one proxy for one account
- Keep proxy details accurate
- Make sure the proxy works before you start trading
- Match the proxy region to your normal account setup if needed

A bad proxy can stop an account from connecting, so check it before you run the bot.

## 🛠️ Common fixes

### App does not open

- Make sure you downloaded the Windows release
- Extract the ZIP file before opening it
- Right-click the app and choose Run as administrator
- Check that Windows did not block the file

### Account will not connect

- Check your login details
- Make sure the internet works
- Test the proxy
- Add the account again if needed

### Bot stops during use

- Close other heavy apps
- Check that the proxy is still live
- Make sure the account has the right settings
- Restart the app and try again

### Buttons look wrong or the window is cut off

- Set Windows display scaling to 100% or 125%
- Maximize the app window
- Restart the app after changing display settings
